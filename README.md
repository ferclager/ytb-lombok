## Notas

Ejecutar el proyecto
```bash
./gradlew bootRun
```

Lanzar tests del proyecto, generando un nuevo jar
```bash
./gradlew clean build
```

Branch inicial
```bash
git checkout main
```

Branch resultado final
```bash
git checkout feat/add-lombok
```
